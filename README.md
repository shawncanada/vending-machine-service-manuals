## 🔧 TROUBLESHOOTING TIPS

### **Common Issues:**

**Vending Machine Won't Power On**
- Check main power connection and breaker
- Verify internal fuses
- Test power supply voltages
- Inspect wiring connections

**Product Not Dispensing**
- Check product delivery system for jams
- Verify motor operation
- Test vend mechanism
- Inspect product sensors

**Refrigeration Problems**
- Check compressor operation
- Verify thermostat settings
- Inspect condenser coils
- Test refrigeration pressures

**Payment System Issues**
- Clean bill validator
- Test coin mechanism
- Check validator alignment
- Inspect cash box sensors

**Display/Control Problems**
- Check controller display
- Test button functionality
- Verify communication cables
- Reset controller if needed
